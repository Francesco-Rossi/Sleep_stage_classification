# Sleep Stage Classification

Questo progetto ha l’obiettivo di sviluppare e confrontare diverse architetture di reti neurali per la classificazione degli stadi del sonno, utilizzando segnali di elettroencefalogramma (EEG) come input.  
Le prestazioni dei modelli implementati vengono valutate e confrontate attraverso diverse metriche, al fine di individuare l’architettura più efficace per il problema considerato.

## Esecuzione del codice

Per eseguire il progetto:

1. Caricare il file `sleep_stage_classification.ipynb` su **Google Colab**.
2. Scaricare il dataset dal sito  
   [TimeSeriesClassification.com – Sleep Dataset](https://www.timeseriesclassification.com/description.php?Dataset=Sleep).
3. Caricare i file `Sleep_TRAIN.ts` e `Sleep_TEST.ts` su Google Colab oppure nel proprio Google Drive.
4. Assicurarsi che i file siano contenuti in una cartella denominata `progetto_applicazioni_ia`.

Una volta completati questi passaggi, il notebook può essere eseguito senza ulteriori configurazioni.
